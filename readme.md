# B12A01 Landing Launchpad

A modern landing page template for Webflow agencies, built with HTML and CSS. This project features a clean UI, engaging hero section, statistics, user feedback, and a contact form.

## Features


- **Hero Section:** Eye-catching headline, call-to-action button, and background.
- **Statistics Section:** Highlights key metrics with bold numbers and labels.
- **User Feedback:** Showcases testimonials with reviewer info.
- **Focus Section:** Encourages users to get started.
- **Contact Form:** Collects user email, name, and message.
- **Footer:** Includes navigation and company info.

## Requirements

- Basic knowledge of HTML and CSS.
- All assets (images, icons) should be placed in the `Assets` folder.
- Use the [Raleway](https://fonts.google.com/specimen/Raleway) font from Google Fonts.
- No JavaScript required.
- Compatible with all modern browsers.

## How to Use

1. **Clone or Download** this repository.
2. Open `index.html` in your browser to view the landing page.
3. Edit the HTML and CSS files to customize content, colors, or layout as needed.
4. Replace images in the `Assets` folder with your own if desired.

## Customization

- **Colors:** Change color codes in `style.css` to match your brand.
- **Text:** Update headings, paragraphs, and button text in `index.html`.
- **Images:** Replace images in the `Assets` folder for your own branding.
